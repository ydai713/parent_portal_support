# Parent Portal - Support

## About

Parent Portal is a simple, modern mobile app that helps school parents stay
organized by gathering off-day calendars, essential school information, events,
resources, and a parent community all in one clean, easy-to-use place.

This early version currently supports one school community. Additional schools
will be added based on parent interest.

## Frequently Asked Questions

### What is this app?
This is a helper app created by a parent to make it easier for families to
access school information, community posts, and stay up to date with school
events.

### Is this an official school app?
No, this is an unofficial app created by a parent for the school community.

### How do I report a bug?
You can report bugs directly in the app via Settings > Report a Bug, or email us
at yang.dai2020@gmail.com.

### Why do some features look empty for new users?
Certain features such as the parent community depend on parent participation.
They will grow as more parents join.
